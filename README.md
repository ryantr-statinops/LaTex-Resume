## 🛠 Cấu trúc dự án
- `/src`: Chứa mã nguồn LaTeX (`main.tex`).
- `/assets`: Chứa tài nguyên hình ảnh (`avt.jpg`).
- `.github/workflows`: Chứa kịch bản tự động hóa CI/CD.
